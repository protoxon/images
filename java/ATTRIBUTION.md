This directory contains Dockerfiles for various Java runtime images used by the SLS project.

These Dockerfiles are **adaptations** of the original images created by the Pterodactyl project and its contributors, including but not limited to:

- Matthew Penner  
- Pterodactyl Software  
- William V. (Infraly)  

The original images and their source code can be found in the Pterodactyl "yolks" repository.

 Changes have been made to tailor these images for SLS, but credit for the original structure and approach belongs to the above creators. These files are maintained by **Jesse Faler** for the SLS project.
